# 项目架构

```
工程架构
    webpack 配置
    node 服务
    服务端渲染

项目架构
    React
    React-Router
    Mobx 配置 代替Redux
    服务端渲染优化

业务开发
    页面开发
    登录服务
    服务端渲染优化

项目部署
    PM2
    Nginx
```

### 项目目标
```
1.工程架构的概念,为什么要搭建架构
2.服务端渲染的难点以及如何解决问题
3.React + React Router + Mobx的项目构架模式
4.掌控项目的全局

```

### 技术区分
```
单页应用(对SEO不太友好，因为网页都是动态的,不一定能抓到好的数据)
多页应用(传统网站,对SEO支持比较友好)
    - 内容都是服务端模板生成的
    - 媒体跳转内容都要经过服务端

```
